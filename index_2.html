🔌 Connect Gemini (Google AI)

Paste your own Google Gemini API key to unlock worksheet reading and progress analysis. It's stored only on this device, in this browser.

Get a key free at aistudio.google.com → API Keys → Create API key. Gemini has a free tier, so you may not need billing. Keep this file private — don't share it with the key saved. Works offline for tracking; reading & analysis need internet.

If the AI steps show a "couldn't reach Google" error: open the file through a local address instead of double-clicking. In the file's folder run python -m http.server and open http://localhost:8000/learning-tracker.html.
